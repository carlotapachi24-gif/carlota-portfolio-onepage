## Archivos

- `styles.css`
- `main.js`

## Cómo verlo en local (recomendado)
**VS Code + Live Server**
1. Abre la carpeta en VS Code
2. Instala la extensión **Live Server**
3. Click derecho en `index.html` → **Open with Live Server**

## Personalización rápida
- Sustituye los placeholders de imagen:
  - Hero: `.ph` (4 bloques)
  - Proyectos: `.thumb` (3 bloques)
- Cambia los links de Behance/LinkedIn/Instagram en el footer (ahora mismo están en `#`)
- Si quieres, puedo añadir:
  - Sección de “Galería” con lightbox
  - Scroll reveal / animaciones
  - Botón de descargar CV (PDF)
